# Programming2_FinalProject
Link to Project: https://github.com/isabellazy/Programming2_FinalProject

Proposed Project Structure:  
multi_db_blast_classifier/  
main.py  
config.py  
01.) create_database.py: generates BLAST databases based on a fasta file of many reference sequences and saves them to local disk

02.) blast_runner.py:  Runs BLAST query of unknown sequence against local database

03.) classifier.py: Interprets BLAST results and ranks hits across different databases  

04.) evaluation.py: Measures accuracy using known sequences and analyzes performance changes
